
## Install Python3 from source code

This script is only available on CentOS, Debian, Ubuntu

Demo as:

```bash
sh -c "$(curl -fsSL https://shell.ismdeep.com/install-python3.sh)" - 3.7.9 /root/python-379
```

Latest Version: `3.7.9`, `3.8.6`, `3.9.0`
